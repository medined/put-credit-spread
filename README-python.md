# Python

## Environment

### Creation

```
conda deactivate
mamba env create --file=environment.yml
```

### Activate

```
conda activate put-credit-spread
```

### Update

```
conda deactivate
mamba env update --file=environment.yml
conda activate put-credit-spread
```

### Deactivate

```
conda deactivate
```

### Delete

```
conda deactivate
conda remove --yes --name=put-credit-spread --all
```
